An easy to use conditional email content plugin. Written the WordPress way.

== Description ==
This plugin adds a shortcode use Email Template content conditionally display in the admin email or any other email template based on the presence of form field data. if some field are not required or optional and you don't want show the static labels and content on the email template then you can use this plugin functionality to achieve this.

== Installation ==

1. Download the plugin.

2. Upload [`conditional-email-content`](conditional-email-content.zip) to the `/wp-content/plugins/` directory or form Wordpress Plugin Menu.

3. Goto the Plugins section of the WordPress admin and activate the plugin.

4. Goto your form email template and use the shortcode to use the conditional funcationality.
